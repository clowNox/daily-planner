# 📋 Daily Planner — AI-Powered Task Priority Manager

A mobile-first daily planning web app that helps you organize your tasks and lets AI decide what to tackle first.

---

## ✨ Features

- **AI Priority Scoring** — Each task is scored 0–100 by Claude AI based on urgency and category. Tasks are ranked High, Medium, or Low priority with a short reason.
- **8 Task Categories** — Work, Essential, Health, Family, Grooming, Pet, Friends, Business — each color-coded for quick scanning.
- **Daily Planning** — Plan tomorrow's to-do list every evening. Add as many tasks as you need throughout the day.
- **Task Completion** — Check off tasks as you complete them. Completed tasks move to a separate "Done" section at the bottom.
- **Progress Tracking** — A live progress bar shows how much of your day you've completed.
- **Filter by Category** — Focus on one area at a time using the category filter bar.
- **Persistent Storage** — Your tasks are saved locally on your device and will be there when you come back.
- **Mobile Optimized** — Designed for phone screens. Add it to your home screen for a native app-like experience.

---

## 🚀 How to Use

1. **Add a task** — Type a task name, pick a due date, and choose a category.
2. **Analyze** — Tap "Analyze with AI" to get priority scores and rankings.
3. **Execute** — Work through your tasks from top to bottom. Check them off as you go.
4. **Re-analyze** — Added new tasks mid-day? Hit "Re-analyze" to refresh the rankings.

---

## 📱 Install as a Mobile App

1. Open the app link in **Safari** (iPhone) or **Chrome** (Android)
2. Tap the **Share** button
3. Select **"Add to Home Screen"**
4. The app will appear on your home screen like a native app

---

## 🛠 Built With

- React (via Babel CDN — no build step needed)

- Vanilla CSS with a native iOS-inspired design
- localStorage for offline data persistence
